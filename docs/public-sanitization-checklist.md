# Public Sanitization Checklist

Before publishing any skill, remove or generalize:

## Company References
- Company names
- Product names
- Internal team names
- Internal project names
- Internal process names

## Customer / Partner References
- Customer names
- Partner names
- Store names
- Brand names
- Account names

## Internal Systems
- Internal URLs
- Azure DevOps links
- Salesforce record links
- Jira links
- SharePoint or Confluence links
- Slack or Teams references

## Sensitive Technical Details
- API endpoints
- Authentication flows
- Environment names
- Internal service names
- Database names
- Credentials, tokens, or keys

## Business-Sensitive Details
- Pricing
- Contract terms
- SOW details
- Revenue estimates
- Forecasts
- Roadmap commitments
- Escalation details

## Replace With
- “enterprise software company”
- “retail platform”
- “CRM system”
- “work tracking system”
- “customer”
- “partner”
- “internal link removed”
- “platform service”
- “example value”
